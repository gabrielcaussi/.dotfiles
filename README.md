# My Dotfiles

My dotfiles with Manjaro bspwm, zsh, kitty, neovim and tmux.

![Desktop](/screenshots/bspwm-gabrielcaussi.png)

```shell
$ sudo pacman -S xorg-xset root 

$ sudo pacman -S xorg-xset bspwm sxhkd kitty rofi feh neovim nautilus flameshot firefox-i18n-pt-br git curl

$ yay -S polybar
```
